# MCM/ICM 2026 Project Repository

## Structure

- **`/code`**: Source code for models, data processing, and analysis.
- **`/data`**: Raw and processed datasets.
- **`/results`**: Output files, figures, and simulation results.
- **`/doc`**: Documentation, notes, and drafts.
- **`/references`**: Academic papers and reference materials.
